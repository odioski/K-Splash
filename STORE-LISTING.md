# KDE Store Listing Draft

## Title
K-Unsplash Widget

## Short Description
Auto-refresh KDE Plasma wallpaper widget powered by the Unsplash API.

## Full Description
K-Unsplash Widget is a Plasma 6 desktop widget that fetches random photos from the Unsplash API and can apply them as your KDE desktop wallpaper.

Features:

- Refresh desktop wallpaper automatically on a user-defined interval
- Choose a category or custom query such as `dark`, `fun`, `skyline`, `happy`, `nature`, or `city`
- Optional wallpaper-changing toggle in widget settings
- Adjustable target image width and height
- Unsplash attribution links shown inside the widget
- Plasma 6 widget settings page for access key, category, and refresh interval

Requirements:

- KDE Plasma 6
- An Unsplash developer access key
- Network access to `api.unsplash.com`

Notes:

- This widget uses the Unsplash API with `Accept-Version: v1`
- The widget package does not include any API secrets
- Users should enter their own Unsplash access key in the widget settings
